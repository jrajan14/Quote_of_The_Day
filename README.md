# Quote of the Day

A sophisticated web application that delivers randomly selected inspirational quotes through an elegant, animated interface.

## Features

- Random quote generation from a JSON database
- Dynamic animated gradient background with smooth color transitions
- 75+ floating animated particles (circles, stars, diamonds)
- Glass morphism design with frosted glass effects
- Smooth fade animations and particle burst effects on quote change
- Fully responsive for desktop, tablet, and mobile
- No dependencies – pure HTML, CSS, and JavaScript

## File Structure

quote-of-the-day/
├── index.html      # Main HTML structure
├── styles.css      # Styling and animations
├── scripts.js      # Quote logic and flying objects
└── quotes.json     # Quote database


## Quick Start

1. Download all four files to the same folder
2. Open `index.html` in any modern browser
3. Click "New Wisdom" for a random quote

## Adding Quotes

Edit `quotes.json` and add new entries:

```json
{
    "text": "Your inspirational quote here",
    "author": "Author Name"
}
```

## Browser Support
Chrome, Firefox, Safari, Edge (all modern versions)
